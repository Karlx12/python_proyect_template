# Hello World Template

This is a simple Hello World template for a Python project.

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

- Python 3.x

### Installation

1. Clone the repository:
  ```sh
  git clone https://github.com/yourusername/python_proyect_template.git
  ```
2. Navigate to the project directory:
  ```sh
  cd python_proyect_template
  ```

### Running the Hello World Script

To run the Hello World script, execute the following command:
```sh
python hello_world.py
```

### Expected Output

```
Hello, World!
```
